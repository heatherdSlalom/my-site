# Heather Davis - Personal Website

A modern, professional personal website showcasing AI coaching expertise and artistic portfolio.

## 📁 Directory Structure

```
my-site/
├── index.html              # Main homepage
├── css/
│   └── styles.css         # All styling and responsive design
├── pages/
│   ├── coaching.html      # AI Coaching page
│   └── art.html           # Art Portfolio page
├── assets/                # Folder for images, icons, media (ready for future use)
└── README.md              # This file
```

## 🎨 Features

- **Modern Design**: Clean, professional aesthetic with gradient accents
- **Fully Responsive**: Looks great on desktop, tablet, and mobile devices
- **Multiple Pages**: 
  - Homepage with hero section and overview
  - AI Coaching page with services and process
  - Art Portfolio page with collections and offerings
- **Color Scheme**: Indigo, pink, and amber gradients with professional typography
- **Easy Navigation**: Sticky header with smooth scrolling
- **Professional Sections**: About, Services, Portfolio, Contact

## 🚀 Getting Started

1. **Open in Browser**: Simply open `index.html` in your web browser
2. **Customize Content**: Edit the HTML files to add:
   - Your profile image (replace "Your Profile Image Here" text)
   - Email address (currently set to `hello@heatherdavis.com`)
   - LinkedIn and social media links
   - Real portfolio/artwork images
   - Testimonials or case studies

## 📝 Customization Guide

### Add Images
1. Create an `images/` folder in the `assets/` directory
2. Add your images there
3. Update HTML references to point to `assets/images/your-image.jpg`

### Update Contact Information
Edit these sections in `index.html`:
- Email: `hello@heatherdavis.com`
- LinkedIn: Add your LinkedIn profile URL
- Social media links

### Change Colors
Edit CSS color variables at the top of `css/styles.css`:
- `--primary-color`: Main brand color (currently indigo)
- `--secondary-color`: Accent color (currently pink)
- `--accent-color`: Secondary accent (currently amber)

### Add New Sections
1. Create new HTML pages in the `pages/` directory
2. Update navigation links in all HTML files
